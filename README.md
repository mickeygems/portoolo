# Portfolio Template

## Features

- Fully Mobile responsive.
- Supports light/dark theme.
- Built with a11y in mind.
- Modern design.
- Multiple sections.
- Sleek animations.
- Fast loads.
- SEO-optimized.



### Cloning the Repository.

If you have [Git](https://git-scm.com/) installed on your computer, you can clone or download the portfolio template by running the following command in your terminal.

```bash
git clone https://github.com/mickeygems/portoolo.git
```

Alternatively, click on this [link](https://github.com/mickeygems/portoolo.git) to download it.
